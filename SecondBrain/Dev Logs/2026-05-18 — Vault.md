## Problems & Solutions

- **Graph RGB values zeroed out by Obsidian** — original values exceeded 24-bit range; recalculated using correct decimal RGB (max 16,777,215). Fixed with: teal `#1ABC9C` = 1752220, blue `#4A90E2` = 4886754, green `#2ECC71` = 3066993.
- **`/obsidian-save` would recreate `Daily/`** — step 5 was hardcoded to write a daily note; patched the command file to append to dev log instead.
- **`/obsidian-log` looked for missing template** — `Templates/Dev Log.md` didn't exist; created it.
- **Empty ghost files** — `Angular.md` created by a bad `[[Angular]]` wikilink in decisions.md; `2026-05-18.md` created by Obsidian's daily note feature at root. Both deleted; wikilink fixed.

## Decisions Made

- **Deleted `Daily/`** — 4 notes were thin wrappers pointing to Dev Logs with no unique content. `Dev Logs/` is now the single session log.
- **Merged `CLAUDE.md` → `_CLAUDE.md`** — two operating manuals collapsed into one. `_CLAUDE.md` is the sole source of truth.
- **`Dev Logs/` is the session log for all work including vault maintenance** — named `YYYY-MM-DD — Vault.md` when no project is involved.
- **`/obsidian-daily` should not be used** — command still installed but vault has no `Daily/` folder; using it would recreate the deleted structure.

## Knowledge Captured

- Obsidian graph color RGB values must be 24-bit integers (0–16,777,215) — larger values get silently zeroed to black.
- Obsidian creates empty ghost files when you click an unresolved `[[wikilink]]` — avoid bare `[[ConceptName]]` links unless a note for them exists or is intended.
